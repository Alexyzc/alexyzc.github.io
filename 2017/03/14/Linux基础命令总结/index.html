Linux基础命令总结

发表于 2017-03-14 | 分类于 Linux | |

开篇

在日常测试、开发过程中，常用到Linux命令，熟练的掌握和使用这些命令可以提高工作效率，对问题的定位也起到事半功倍的效果。

Linux命令篇（基础命令）

  • vi编辑器：

    • 功能：生成新文件或编辑查看文件
    • 格式：vi file_name（文件名称）
    • 说明：上面的命令直接进入vi的命令模式
      从命令模式进入的编辑模式需要在当前键盘上按“i”或“a”键，使用ESC键可以从编辑的模式进入命令模式。
    • 编辑模式可执行的操作：插入各种文本信息
    • 编辑模式向命令模式转换，可以按下面的一种键：
      • “i”：插入文本
      • “a”: 追加文本
    • 命令模式可以执行的操作
      • “:w”: 保存文本
      • “:wq”: 保存并退出
      • “:wq!”: 保存并强制退出
      • “:q”: 退出
      • “:q!”: 强制退出
      • “dd”: 删除一行文字(当前删除行，点击键盘dd)
      • “x”：删除一个字符
      • “:n”: 光标移至文本第n行
      • “$”: 光标移到文本的行尾
      • “G”: 光标移到文本的末尾
      • “ZZ”: 存盘退出
      • “/字符串”: 查找某个字符串
    • 命令模式向编辑模式转换，可以按ESC键

  • 系统开关及重启

  • pwd用法

    • 功能描述：查看用户当前所处的绝对路径

  • ls用法

    • 功能描述：显示指定目录下的内容
    • 格式 ：ls [参数] [目录或文件]

  • cd用法

    • 功能描述：改变当前目录
    • 格式 ：cd 目录

  • mkdir&rmdir用法

    • 功能：创建目录
    • 格式：mkdir[参数][目录名1][目录名2]

  • cp用法

    • 功能：拷贝文件或目录
    • 格式：cp[参数][源文件或目录][目标文件或目录]
      cp命令总结

  • Mv用法

    • 功能：可以对文件或目录进行移动
    • 格式：mv[参数][源对象][目的对象]

  • rm用法

    • 功能：删除文件或删除非空目录
    • 格式：rm [参数]文件或目录

  • chmod用法

    • 功能：修改文件权限
    • 格式：chmod [参数] 权限 文件或目录
      chmod详解

  • chown用法

    • 功能：修改文件的属主（该例中的组wuxh需要替换为你系统中存在的组）
    • 格式：chown [参数] 用户名 文件

  • chgrp用法

    • 功能：修改文件的所属组
    • 格式：chgrp [选项] 组名 文件

  • find用法

    • 功能：从指定的目录下开始查找某个文件
    • 格式：find 起始目录 寻找条件 文件名

  • grep用法

    • 功能：从某个文件中查询指定的字符串
    • 格式：grep [选项] [查找模式] [ 文件名1，文件名2…]

  • echo用法

    • 功能：在显示屏上回显文字或者用来查看环境变量的值

  • su用法

    • 功能：切换用户 注意su 与su - 的区别

  • clear用法

    • 功能：清屏

  • man用法

    • 功能：联机帮助

  • help用法

    • 功能：在线命令查看帮助

  • whereis用法

    • 功能：查看命令的存放文职

  • id 用法

    • 功能：查看当前用户的uid、gid、groups

  • useradd与userdel

    • 功能：添加用户

  • passwd用法

    • 功能：修改用户口令，注意，如果当前用户是root，该命令修改的将是root的口令

  • groupadd 用法

    • 功能：添加组

  • ps用法

    • 显示进程

  • top命令

    • 动态显示进程信息
      [root@localhost ~]# top

  • kill命令

    • 功能：将指定的进程杀死
    • 格式：kill -s 信号名称 pid / kill -n 36 pid

  • tar命令

    • 功能：将文件或目录进行打包
    • 格式：tar[参数] 打包后的文件名 文件或目录

  • gzip命令

  • rpm命令

  • ifconfig命令

  • route命令

    • 功能：添加路由和查看路由信息

  • netconfig命令

    • 功能:实现网络的配置
